城市医院用血实时联网平台简介

![](图片6.jpg)
       城市医院用血实时联网平台是一套实现城市级医院的临床用血信息和实时联网管理的系统，并对临床用血情况进行科学有效分析和指导。

      将血液全过程质量管理延伸到临床用血，实现对全市临床用血信息的全方位监控。

<!-- Slide number: 3 -->
血管到血管

全市血液信息联网共享和业务协同，形成完整的“血管到血管”的信息链，实现采供血与临床用血环节一体化管理。
血液追溯，保障血液安全
贯彻“合理用血”理念，避免血液浪费
统筹管理和监控，优化全市血液库存

安装简介

docker-compose安装
把docker-compose文件夹上传到/data目录下面
cd /data/docker-compose/
1 复制docker-compose安装包到 /usr/local/bin  下
cp docker-compose /usr/local/bin
2 添加可执行权限
chmod +x /usr/local/bin/docker-compose
3测试安装结果
docker-compose --version
返回以下结果就是安装成功
docker-compose version 1.23.1, build 1719ceb
把docker-offline 上传到/data目录下面
cd /data/docker-offline/
tar xf docker-17.03.1-ce.tgz
cp docker/docker* /usr/bin
cp docker.service
/usr/lib/systemd/system/docker.service
systemctl enable docker
systemctl start docker
docker -v
出现下面字符代表docker离线安装完成
Docker version 17.03.1-ce, build c6d412e

业务系统配置

设置接口账号（用于医院联网血站端--机构管理--医院联网账号）

1.账号授权管理--权限管理 添加医院联网角色 具备操作权限的功能菜单 勾选医院联网和成品库的预定管理

账号授权管理--用户管理 添加医院用户 账号类型选择 “医院用户”  角色选择刚才创建的角色

2.字典维护--用血单位--用血单位 找到所属的医院设置医院联网账号
